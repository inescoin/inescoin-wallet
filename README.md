# Inescoin Blockchain - Offline Wallet

Clone project
```
  git clone git@github.com:inescoin/inescoin-wallet.git
  cd inescoin-wallet/
```

Install dependencies
```
  npm i

  # After installation you need run patch
  node patch.js
```

Run with dev env
```
  npm start

  # open browser => http://localhost:4200
```
